## 搭建hugo

### 环境

- hugo
	- `sudo pacman -S hugo`

### 初始化

```
hugo new site name
```

- Create a new site in the provided directory. The new site will have the correct structure, but no content or theme yet.

### 下载主题

```
git submodule add https://github.com/guangsizhongbin/LoveIt themes

```
