安装Raspberry Pi OS

下载系统

系统下载地址Raspberry Pi OS

烧录镜像

  1. sudo fdisk -l

先查看SD卡的名称

sudo dd bs=4M if=系统路径 of=/dev/SD名称 status=progress oflag=sync

Tip

  1. 使用dd时，写SD卡的名称，不要写SD卡分区的名称

  2. 使用dd时，报错 d: opening /dev/sdc: Read-only file system

  3. dd参数

  • bs=BYTES read and write up to BYTES bytes at a time (default:512); overrides ibs and obs

  • status=LEVEL The LEVEL of information to print to sederr; none suppresses everything but error messages, noxfer suppresses the final transfer statistics, progress shows periodic transfer statistics.

  • nocache Request to drop cache. See also oflag=sync

开启SSH，并连接

  1. SD卡的boot，创建名为ssh`的文件即可

  2. ssh pi@pi_ip_address

默认密码raspberry

下载并配置samba

  1. sudo apt install samba samba-common-bin // 下载samba

  2. sudo apt install vim // 下载vim

  3. sudo vim /etc/samba/smb.conf //配置samba

  1. testparm //测试是否通过

  2. sudo smbpassed -a pi //smb加入pi用户

  3. systemctl enable smbd //开机启动smbd

  4. systemctl start smbd //启动smbd

下载并配置aria2

  1. sudo apt install nginx

  2. sudo apt install aria2

  3. cd /var/www/html

  4. sudo wget https://github.com/mayswind/AriaNg/releases/download/1.1.7/AriaNg-1.1.7-AllInOne.zip

  5. unzip AriaNg-1.1.7-AllInOne.zip

  6. systemctl start nginx //启动nginx

  7. systemctl enable nginx //开机启动nginx
