什么是类? 什么是对象?

类:

表示一个客观世界中某类群体的一些基本特征抽象

如:

汽车， 轮船，书

对象:

一个个具体的事物

如:

张三同学， 李四账户， 王五的汽车

总结:

类 实际上是对象操作的 模板 ，但是类不能够直接使用，必须通过 实例对象 来使用

在 java 中什么是堆内存 (heap) 和 栈内存 (stack) ?

栈内存 :

保存的是一块对内存的空间地址

堆内存 :

保存对象的真正数据, 都是每一个对象的属性内容
