Convert2

1
2
3
4
5

# convert single page of PDF file to image
convert presentation.pdf[0] test.jpg

# convert from page 0 to page 5
convert presentation.pdf[0-5] test.jpg
Note

  • -density used to specify the DPI of the output images.

  • quality specify the quality for the generated images.

  • %3d used to specify the format for generated image names. The generated images will be named output-001.jpg, output-002.jpg
