什么是异常? 异常 程序运行时期出现了问题，出现了不正常的情况，称之为 异常 什么是编译出错? 编译出错 在程序的编译时期不能通过编译器的语法检查，所以报错 什么是异常类? java 当中一切皆对象， 当程序产生异常， JVM 会把这个异常信息封装成一个对象(类) 类中封装着问题的名称, 产生的原因， 描述等多个属性信息存在 以及对这些信息进行操作的一系列方法 这些类通过继承层次构成了 java 的异常体系 异常中有什么需要注意的? 异常的类和对象中只是存了异常的信息 包括异常的原因，异常的种类等等 何时抛出异常，怎么处理异常，不是由异常对象决定的 java 中对异常体系是如何划分的? Throwable (祖先类) Error Exception RuntimeException 非RuntimeException Throwable 是 java 一切错误和异常的父类，是继承层次中的祖先类 Error 是严重问题， 无法被解决 - Error 描述了 java 运行时虚拟机内部错误和资源耗尽错误 对于 Error , 程序自己是无能为力的，仅靠程序本身是无法恢复和预防
